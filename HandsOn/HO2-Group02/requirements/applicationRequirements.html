SafeSchool Madrid - Application Requirements

General Description

SafeSchool Madrid is a web application designed to map road accidents and speed cameras, and to assess safety risks around schools (within 150–300 meters). The app highlights "priority zones" where interventions such as new speed cameras, speed bumps, pedestrian crossings, or human supervision are necessary. Main outputs include interactive maps, school risk rankings, informative profiles, and exportable reports.

Selected Datasets

  • Educational Centers of the Madrid Community: geographic information and characteristics of schools.
    Dataset link
  • Road Accidents in Madrid: data on accidents with geographic location and severity.
    Dataset link
  • Fixed and Section Speed Cameras: data on speed control devices.
    Dataset link

Functional Requirements

  • Interactive map of schools with accident and speed camera overlays.
  • Risk calculation for each school within 150–300 meters, highlighting priority zones.
  • Ranking of the most at-risk schools.
  • Informative profiles for each school, including accident data, nearby speed cameras, and other safety notes.
  • Export of data in PDF or CSV reports.
  • Filters by school type, neighborhood, and accident severity.
  • Ability to link RDF datasets for advanced analysis and semantic queries.

Non-Functional Requirements

  • Responsive web interface compatible with desktop and mobile devices.
  • Map loading time within 3 seconds.
  • Data security with admin-only access.
  • Scalable architecture to handle real-time dataset updates.
  • Multilingual support (initially English and Spanish).
